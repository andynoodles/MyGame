# Pacman Clone
利用PTSD框架重製Pacman


## Getting Start
```
git clone https://github.com/andynoodles/MyGame --recursive
```

## Build Game
Using Linux/MacOS
```
cmake -DCMAKE_BUILD_TYPE=Debug -B build && cmake --build build  && ./build/pacman
```
Using Windows
```
cmake -DCMAKE_BUILD_TYPE=Debug -B build ^ cmake --build build  ^ ./build/pacman
```
